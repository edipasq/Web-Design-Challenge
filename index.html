Summary: Latitude vs X

On this project we are analizing how weather changes as you get closer to the equator. The data was pulled from OpenWeatherMap API dataset on over 500 cities. To be exact this information was pulled on Monday, ‎September ‎20, ‎2021, ‏‎9:22:27 PM.

After retrieving the data we used Matplotlib to plot different aspects of the weatherin comparison to Latitude. We looked at temperature, cloudiness, humidity and wind speed. We will provide you the data of the 545 Cities used in this project as well as visualizations that are part of this project. We will also show comparisons amont all the weather factors that were compared with the latitude.
